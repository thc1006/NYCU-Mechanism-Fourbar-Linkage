## 這個題目的重點是什麼？

整個作業在做的事可以拆成幾個步驟：

1. **看影片，理解機構構造**
   把影片裡的那個「智能櫃機構」看懂，它的零件怎麼連動、怎麼開關門、怎麼取放東西。

2. **依照老師限制畫出構造圖（結構圖）**
   在老師給的條件／限制（例如：尺寸上限、可用材料、機構形式）之下，把原本的機構畫成標準的構造圖（structure diagram），而不是只畫草圖。

3. **把幾何尺寸變成「有數字」的設計**
   要算清楚／標出來：

   * 長度（length）
   * 寬度（width）
   * 深度（depth）
     然後由這些數值去算出整個櫃子的「容量空間」（volume / capacity）。

4. **重新設計一個「容量更大」的智能櫃機構**
   在老師給的限制內，用自己的構想重新畫一個「容量空間更大」的 **智能櫃（smart cabinet / smart locker）** 機構（mechanism），不是亂放大，而是有邏輯、有理由的優化設計。

5. **要有「數學解析＋軟體模擬」**

   * 數學上的解析（mathematical analysis）：例如幾何關係、容量公式、機構運動範圍等等。
   * 軟體模擬圖（software animation / simulation）：用軟體把機構建起來，看它「實際動起來」是什麼樣子。

整體來說，這是一個 **大學部（undergraduate-level）機構設計作業**，同時訓練你：看機構影片→畫圖→算尺寸容量→優化設計→數學說明→軟體模擬。

## 修正後逐字稿（含中英專有名詞）

> 就是要先看那個影片，然後了解它的構造（構造／結構：structure），接著要按照老師給的限制條件（constraints），去畫出它的構造圖（structure diagram），然後把這個構造是用哪些數值（parameters）去定義的解出來，也就是它的長度（length）、寬度（width）、深度（depth），再由這些去算出它的容量空間（volume / capacity）。之後，再用自己的設計圖，去設計一個容量空間更大的「智能櫃」（smart cabinet / smart locker）機構（mechanism）。最好還要有數學上的解析（mathematical analysis），以及用軟體（software）做出它實際上動起來時樣子的圖，也就是模擬動畫（animation / simulation）。這是一個大學部（undergraduate-level）的作業題目。
