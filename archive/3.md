### 圖片整體分析

這張筆記似乎在求解一個向量系統。開頭的向量圖（左上角）顯示了三個位置（$P_1, P_2, P_3$），它們似乎是由兩個向量 $W$ 和 $Z$ 的不同旋轉組合相加而成（$W_i + Z_i = R_i$）。

筆記的主要內容是將這些向量方程式轉換為複數形式（例如 $W e^{i\beta}$），然後嘗試通過分離實部（Real part, RL.）和虛部（Imaginary part, Im.）來解出一系列的未知數。

筆記中提到有 5 個未知數（`Z, W, \phi, r_{1x}, r_{1y}`），並嘗試建立 5 個有用的方程式來求解。推導過程中充滿了代數和三角函數的恆等式變換。

### 1\. 左上角：向量圖

這是一個向量加法圖。

  * 有三個 $W$ 向量（$W_1, W_2, W_3$），它們有共同的起點和可能相同的模長 $W$，但角度不同（$\beta_1, \beta_2, \beta_3$）。
  * 有三個 $Z$ 向量（$Z_1, Z_2, Z_3$），它們分別接在 $W$ 向量的末端，可能有相同的模長 $Z$，但角度不同（$\phi_1, \phi_2, \phi_3$）。
  * $R$ 向量（$R_1, R_2, R_3$）是 $W$ 和 $Z$ 向量的和（$R_i = W_i + Z_i$），代表從原點到 $P$ 點（$P_1, P_2, P_3$）的最終位置向量。
  * 圖中還標記了角度 $\theta$、$\phi$、$\beta_1$、$(\phi_1 - \beta_1)$ 以及 $\vec{u_1}$, $\vec{u_2}$ 單位向量。

### 2\. 右上角：基本方程式

這是問題的數學模型設定。

```
sol. thus it can be derived
    { W_1 + Z_1 = R_1
    { W_2 + Z_2 = R_2
    { W_3 + Z_3 = R_3
        ->
    { W e^(i\beta_1) + Z e^(i\phi_1) = R_1
    { W e^(i\beta_2) + Z e^(i\phi_2) = R_2
    { W e^(i\beta_3) + Z e^(i\phi_3) = R_3

let W = W e^(i\theta) & Z = Z e^(i\phi).  \beta_i, \phi_i, R_i known
```

  * **分析**：這裡將向量加法 $W_i + Z_i = R_i$ 轉換為複數形式。$W$ 和 $Z$ 被假定為模長（magnitudes），$\beta_i$ 和 $\phi_i$ 是它們各自的角度。$R_i$ 是已知的結果向量。
  * `let W = W e^(i\theta) & Z = Z e^(i\phi)` 這一行似乎是在設定 $W$ 和 $Z$ 的一般複數形式，其中 $W$ 和 $Z$ 是模長，$\theta$ 和 $\phi$ 是它們的基底角。

### 3\. 右上角（第二區塊）：簡化的方程式

這似乎是通過旋轉座標系（例如，讓 $Z$ 向量對齊實軸）來簡化問題。

```
-> { W + Z = R_1
   { W(cos\alpha + i sin\alpha) + Z = R_2
   { W(cos\beta + i sin\beta) + Z = R_3
```

  * **分析**：這個轉換假設 $Z$ 向量沒有虛部（即 $\phi=0$），所以 $Z$ 是一個實數。$W$ 向量的角度變為 $\alpha$ 和 $\beta$（這些可能是 $\beta_2 - \beta_1$ 和 $\beta_3 - \beta_1$ 的簡寫）。

### 4\. 中間左側：一般形式與劃掉的嘗試

這一區塊顯示了更一般的代數形式，以及一個被劃掉的錯誤嘗試。

```
=> { W + Z = R_1
   { W e^(i\alpha) + Z e^(i\gamma) = R_2
   { W e^(i\beta) + Z e^(i\delta) = R_3

   [...以下是被 X 劃掉的內容...]
   { W + Z = R_1
   { W(i) + Z = R_1 + P_Z1
   { W(j/i) + Z = R_1 + P_i
```

  * **分析**：上面未劃掉的區塊是右上角（第二區塊）的更一般形式，允許 $Z$ 向量也具有旋轉角度 $\gamma$ 和 $\delta$。被劃掉的部分似乎是一個錯誤的推導步驟。

### 5\. 中間右側：混亂的推導（實部與虛部）

這部分非常混亂，似乎是將某個方程式分離實部和虛部時的草稿，有標註 (a), (b), (c), (d)。

```
RL.
W cos\theta + Z cos\phi = r_1x
W(i) sin\theta + Z sin\phi = r_1y - x = r_1x - \sqrt{i} W cos 45°    (a)
W/2(cos\theta + i sin\theta) + Z(cos\phi + i sin\phi) = r_1y - x = r_1x - W \sqrt{2} cos 22.5°   (b)

Im.
W sin\theta + Z sin\phi = r_1y
W(i) cos\theta + Z sin\phi = r_1y - y = r_1y - \sqrt{i} W sin 45°    (c)
W/2(cos\theta + i sin\theta) + Z(cos\phi + i sin\phi) = r_1y - y = r_1y - W \sqrt{2} sin 22.5°   (d)
```

  * **分析**：這部分是從 $W e^{i\theta} + Z e^{i\phi} = R_1 = r_{1x} + i r_{1y}$ 出發。
  * `RL.` (實部): `W cos\theta + Z cos\phi = r_1x`
  * `Im.` (虛部): `W sin\theta + Z sin\phi = r_1y`
  * 下面的 (a), (b), (c), (d) 行似乎是其他方程式（可能是 $R_2$ 或 $R_3$）的實部和虛部，並且寫得非常潦草。`W(i)` 的寫法不標準，可能是 $W/2$ 或 $W \cdot i$ 的筆誤。$\sqrt{i}$（根號 $i$）是一個複數 $(1+i)/\sqrt{2}$。

### 6\. 中間左側（下方）：求解的關鍵

這段文字說明了求解的目標和條件。

```
=> since \theta - 90° only 5 eqns are useful, and can solve 5 unknowns  Z, W, \phi, r_1x, r_1y
```

  * **分析**：這是一個關鍵說明。它指出在 $\theta = 90^\circ$ 的特定條件下（`\theta - 90°` 可能是 $\theta=90^\circ$ 的筆誤，或者 $\theta$ 與某個基準相差 90 度），只有 5 個方程式是有效的，剛好可以用來解 5 個未知數：$Z$ (模長), $W$ (模長), $\phi$ ($Z$ 的角度), $r_{1x}$ ($R_1$的x分量), $r_{1y}$ ($R_1$的y分量)。

### 7\. 左下角：分離實部與虛部（最終版）

這是筆記中最清晰的推導結果，似乎是基於 $\theta=90^\circ$ 的假設，將方程式分離為實部和虛部。

```
RL.
{ W cos\theta + Z cos\phi = r_1x
{ -W + Z cos\phi = r cos\sigma - W
{ -W/2 + Z cos\phi = r cos\sigma - W(\sqrt{3}/2)

Im.
{ W + Z sin\phi = r sin\sigma
{ Z sin\phi = r sin\sigma - W
{ W/2 + Z sin\phi = r sin\sigma - W(\sqrt{3}/2)
```

  * **分析**：
      * `RL.` (實部) 和 `Im.` (虛部)
      * 第一行 `W cos\theta + Z cos\phi = r_1x` 和 `W + Z sin\phi = r sin\sigma`。注意 `Im.` 的第一行 `W` 前面沒有 $\sin\theta$，這強烈暗示 $\sin\theta = 1$，即 $\theta = 90^\circ$。這與 `RL.` 中的 `cos\theta = 0` (所以 `W cos\theta` 項消失) 相符。但筆記中 `RL.` 第一行卻寫了 `W cos\theta`，這有點矛盾。不過，如果 $\theta = 90^\circ$，`RL.` 第一行應為 `Z cos\phi = r_1x`，`Im.` 第一行應為 `W + Z sin\phi = r_{1y}` (筆記中的 $r \sin\sigma$ 可能就是 $r_{1y}$)。
      * 後面的方程式引入了 $r$ 和 $\sigma$ (可能是 $R_2$ 或 $R_3$ 的極座標)，並使用了 $W/2$ 和 $W(\sqrt{3}/2)$，這暗示了 $\cos(60^\circ)$ 和 $\sin(60^\circ)$ 或 $\cos(30^\circ)$ 和 $\sin(30^\circ)$ 的值，說明 $R_2$ 和 $R_3$ 的角度可能是 30 或 60 度。

### 8\. 右下角：小輔助圖

這是一個小的三角函數圖形。

```
\psi = 22.5
```

  * **圖形**：一個直角三角形，底角為 $45^\circ$。從頂點畫出一條線，與水平線成 $67.5^\circ$ 角。
  * **分析**：$90^\circ - 67.5^\circ = 22.5^\circ$。這個 $22.5^\circ$ (即 $\psi$) 與 $45^\circ$ 的出現，與中間右側混亂草稿中的 `cos 45°` 和 `cos 22.5°` / `sin 22.5°` 相對應，說明那部分的推導與這個角度有關。
