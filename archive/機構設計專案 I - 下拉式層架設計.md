這是一個大學機構設計課程的專案作業，要求設計一個使用四連桿機構（four-bar mechanism）的下拉式層架系統，安裝於壁掛式櫃子中。

## 設計需求與限制條件

### 空間限制
- **櫃子內部空間**: 80 cm (高) × 40 cm (深)
- **櫃子底部高度**: 距離地面 170 cm
- **層架最低高度要求**: 至少下降至 140 cm（下降至少 30 cm）

### 設計目標
1. 使用四連桿機構（four-bar mechanism）
2. 層架初始時必須完全收納於櫃子內部
3. 層架可被拉出並下降至至少 30 cm 以下
4. 最大化儲存空間（rectangular area）

## 設計方法論摘要

### 從手稿分析中提取的關鍵技術

#### 1. 固定支點位置的機構合成 (Synthesis for Specified Fixed Pivot Location)
- **適用條件**: O₂ 和 O₄ 固定在同一 ground link (link 1) 上
- **關鍵假設**: Link 4 的姿態在運動過程中保持不變（如保持水平）
- **幾何關係**: 兩向量間的角度差 κ 為常數

#### 2. 運動範圍設計
- **目標**: 容器下降距離 = 30 cm
- **轉角限制**: 關鍵連桿相對轉角 Δφ ≈ 40°
- **數學工具**:
  - 畢氏定理（Pythagorean theorem）
  - 餘弦定律（Law of cosine）
  - 向量分析（Vector analysis）
  - 複數方程式（Complex number equations）

#### 3. 空間約束條件
- 容器外緣與櫃體內緣距離 ≥ d (d > 0)
- 固定支點座標需預先指定: (x₂, y₂), (x₃, y₃)
- 所有座標值 x, y > 0（位於第一象限）

#### 4. 機構安全設計
- 設計特殊形狀的限制元件（stopper/cam/lug）
- 防止容器停止後機構繼續旋轉
- 軸承組件（bearing components）的配置

#### 5. 精度提升策略
- 增加中間姿態點（mid-point）作為設計約束
- 在關節 3-4-5 軌跡中選擇關鍵點
- 多指定精確位置可讓實際運動更接近需求曲線

## 數學模型要點

### 向量迴路方程式（Vector Loop Equation）
```
W₁ + Z₁ = R₁
W₂ + Z₂ = R₂
W₃ + Z₃ = R₃
```

### 複數形式表示
```
W e^(iθ) + Z e^(iφ) = R₁
W e^(i(θ+φ₁)) + Z e^(i(φ+α₁)) = R₂
W e^(i(θ+φ₂)) + Z e^(i(φ+α₂)) = R₃
```

### 特殊條件
當 θ - φ = 90° 時：
- 只有 5 個方程式有效
- 可求解 5 個未知數: Z, W, φ, r₁ₓ, r₁ᵧ

### 分離實部與虛部
**實部 (Real part)**:
```
W cos θ + Z cos φ = r₁ₓ
-W + Z cos φ = r cos σ - W
-W/2 + Z cos φ = r cos σ - W(√3/2)
```

**虛部 (Imaginary part)**:
```
W sin θ + Z sin φ = r₁ᵧ
Z sin φ = r sin σ - W
W/2 + Z sin φ = r sin σ - W(√3/2)
```

### 幾何關係
```
|w₁| = |w₂| = W
W = R₁/√2  (由畢氏定理)
S₂ = 225°
β₃ = 45°
β₃² = 2W²(1 - cos 45°) = W²(1 - √2)
β₁ = W√(1 - √2)  (由餘弦定律)
```

## 設計考量重點

### 機構類型選擇
考慮使用的四連桿機構類型：
- **曲柄搖桿機構** (Crank-rocker mechanism)
- **雙曲柄機構** (Double-crank mechanism)
- **雙搖桿機構** (Double-rocker mechanism)

選擇依據：
- 運動範圍需求
- 空間限制
- 力學優勢
- 穩定性考量

### 儲存空間最大化策略
1. **垂直空間利用**: 最大化層架表面積
2. **連桿配置優化**: 減少連桿對空間的侵佔
3. **折疊機制**: 考慮多層或可折疊設計
4. **邊界優化**: 在安全間隙 d 最小化的前提下最大化使用空間

### 安全性考量
- 防止過度旋轉的限制機構
- 承重能力計算
- 穩定性分析
- 止擋設計（stopper design）
